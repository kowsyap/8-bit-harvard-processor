8-bit logic and unsigned fixed point arithmetic circuit.

It contains following circuits:
  -recursive dabling based adder (add,subtract)
  -wallace tree multiplication (multiply)
  -non restoring based division (divide)
  -barrel shifter (logical shift left,right)
  -rotate left
  -not
  -and
  -or
  -nand
  -nor
  -xor
  -xnor
  -greater than comparision
  -equal to comparision
